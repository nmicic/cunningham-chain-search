# Interactive Visualizations

Standalone HTML/JS tools. No build step or server needed — open locally or use the GitHub Pages links below.

## Live Demos (GitHub Pages)

| Tool | Description |
|------|-------------|
| [Chain Mesh](https://nmicic.github.io/cunningham-chain-search/visualizations/chain-mesh/) | 2-adic square-perimeter map with CC1/CC2 edges |
| [2-Adic Tree](https://nmicic.github.io/cunningham-chain-search/visualizations/2adic-tree/) | 2-adic tree with inspectable local chain structure |
| [3D Fold](https://nmicic.github.io/cunningham-chain-search/visualizations/3d-fold/) | 3D shell structure across levels |
| [p+1 Analysis](https://nmicic.github.io/cunningham-chain-search/visualizations/p1-analysis/) | Factor coloring for primes, mod-p view for composites |
| [Chain Analyzer](https://nmicic.github.io/cunningham-chain-search/visualizations/chain-analyzer/) | Single-chain analysis, breaker autopsy, residue immunity |
| [Campaign Dashboard](https://nmicic.github.io/cunningham-chain-search/visualizations/campaign-dashboard/) | Campaign tracking across bit ranges and chain lengths |
| [Immunization Dashboard](https://nmicic.github.io/cunningham-chain-search/visualizations/immunization-dashboard/) | Residue immunity analysis |

To self-host: Settings > Pages > Source: deploy from branch, folder: `/ (root)`.
