# Interactive Visualizations

Standalone HTML/JS tools. Open any `index.html` directly in a browser — no build step or server needed.

| Tool | Description |
|------|-------------|
| [chain-analyzer](chain-analyzer/) | Single-chain analysis, breaker autopsy, residue immunity |
| [2adic-tree](2adic-tree/) | 2-adic tree with inspectable local chain structure |
| [3d-fold](3d-fold/) | 3D shell structure across levels |
| [p1-analysis](p1-analysis/) | Factor coloring for primes, mod-p view for composites |
| [chain-mesh](chain-mesh/) | 2-adic square-perimeter map with CC1/CC2 edges |
| [campaign-dashboard](campaign-dashboard/) | Campaign tracking across bit ranges and chain lengths |
| [immunization-dashboard](immunization-dashboard/) | Residue immunity analysis |

These tools are compatible with GitHub Pages. To enable: Settings > Pages > Source: deploy from branch, folder: `/ (root)`.
